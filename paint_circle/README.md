# Paint Circle

Draw circles on a matrix with an interactive Python program.

## Quick Start

1. **Run the program:**
   ```bash
   uv run python circle_test.py
   ```

2. **Follow the prompts:**
   - Enter radius (1-20)
   - Choose matrix size (default: 50)
   - Pick center coordinates
   - Select display method

3. **See your circle!**
